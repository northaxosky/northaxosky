# Kuzey Gök

Software Development Engineer at Microsoft. CS graduate. Previously interned at AWS.

I build across a few lanes: full-stack web apps, developer tooling, game engine experiments, and Fallout 4 reverse-engineering/modding. Most of my side projects start from a practical itch: make something faster, automate a workflow, improve a tool, or learn a system deeply enough to bend it.

## What I work on

- **Product & web apps** - Next.js/React/TypeScript apps, dashboards, portfolio work, authentication, data visualization, and polished UI systems.
- **Finance tooling** - Vault, a personal finance dashboard with Plaid integrations, budgets, analytics, recurring charge detection, alerts, and secure token handling.
- **Game tech** - Kuma, a lightweight C++ game engine with Vulkan rendering, SDL3 platform code, resource loading, input, timing, and a growing core architecture.
- **Fallout 4 modding** - F4SE plugins, runtime patching, crash fixes, performance work, reverse engineering, and compatibility across OG/NG/AE runtimes.
- **Developer workflow** - CLI tools, workflow experiments, automation, and small utilities that make daily development less annoying.

## Current focus

- Growing **Kuma** from a renderer/platform layer into a usable small-game engine.
- Maintaining and experimenting with **Fallout 4 engine/plugin work**, including performance patches and quality-of-life mods.
- Improving my local developer environment and terminal-first workflow.
- Building and hardening **Vault** as a real personal finance platform.
